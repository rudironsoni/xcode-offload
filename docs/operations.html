Common operations

These flows cover setup, mount verification, Apple-tool checks, and rollback.

Install on a new machine

  1. Mount the external volume.
  2. Set XCODE_OFFLOAD_ROOT to a path on that volume.
  3. Run repair --dry-run and read the human-readable plan.
  4. Run user repair with --load.
  5. Install the system daemon with sudo.
  6. Run doctor --strict.
export XCODE_OFFLOAD_ROOT="/Volumes/YourExternalVolume"
xcode-offload repair --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope user --install-shims --dry-run
xcode-offload repair --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope user --install-shims --load
sudo xcode-offload daemon install --root "$XCODE_OFFLOAD_ROOT" --home "$HOME"
xcode-offload doctor --root "$XCODE_OFFLOAD_ROOT" --require-shims --strict

Enable APFS mount mode

xcode-offload mounts install --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope user --dry-run
xcode-offload mounts install --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope user --load
sudo xcode-offload mounts install --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope system --load
xcode-offload mounts status --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope all

Normal output is intentionally short. Add --verbose to install, repair, uninstall, or status commands when you need the raw command list or every individual mount check.

Verify in a scratch root

xcode-offload mounts verify \
  --scratch-root "/Volumes/YourExternalVolume/xcode-offload-verify" \
  --mode user

The command creates disposable sparsebundles, mounts them, checks status, unmounts them, and removes the scratch run directory.

Run Apple tools without shims

PATH="/usr/bin:/bin:/usr/sbin:/sbin" /usr/bin/xcrun simctl list devices available
PATH="/usr/bin:/bin:/usr/sbin:/sbin" /usr/bin/xcodebuild -version

If APFS mount mode is working, these commands should see the mounted Apple paths without shim involvement.

Back out user mount mode

xcode-offload mounts uninstall --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope user --unload

Back out system mount mode

sudo xcode-offload mounts uninstall --root "$XCODE_OFFLOAD_ROOT" --home "$HOME" --scope system --unload
